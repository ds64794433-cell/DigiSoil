# 🛡️ DigiSoil'26
**Department of Civil Engineering | MITS-DU Gwalior**

DigiSoil is a Python-based interactive application designed to automate soil index property tests and classify soil according to **Indian Standard (IS:1498)**.

## 🚀 Features
- **GSD Module:** Automates Sieve Analysis and plots the Grain Size Distribution curve.
- **Atterberg Module:** Calculates Liquid Limit, Plastic Limit, and Plasticity Index.
- **Classification:** Automatically identifies Soil Symbols (e.g., CH, SW, CL-ML).
- **Reporting:** Generates a comprehensive PDF report with data tables and charts.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Libraries:** Pandas, NumPy, Matplotlib, FPDF2
